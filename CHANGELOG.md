All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-24

### Added

- Initial public release of the Linear Issue AI Raycast extension.
- AI-powered parsing of selected text + user context into structured JSON.
- Linear GraphQL integration to create issues with team/project/cycle resolution.
- Dual AI provider support (Raycast AI or OpenAI).
- Basic documentation and MIT license.
